This website will parse your conversation log and display various informations from it. Everything happens inside your browser, no information will be seen nor collected by us. Also no registration is needed.

How does it work?

When you upload a chat history text file your browser will parse each message and display various statistics.

Will my chat history be stored or shared with anyone?

No. Your chat history will only be read by your browser. It will not be sent anywhere.

The site is not working for me. What can I do?

Recommended browsers are Google Chrome and Mozilla Firefox. If the issue still persists you can submit an issue on GitHub. Please include as much information as possible. (Browser type and version, browser and LINE language setting etc.)

I have an idea which can improve the site. What can I do?

Feel free to contact me, create a new issue or pull request on GitHub. See the contact section for more information.

If you notice any problems or have any comments about the site feel free to contact me by email or create a new issue on GitHub.
